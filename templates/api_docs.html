API Documentation

Use our API to manage your bots programmatically.

Authentication

Authentication is handled via a user-specific API key. You must include your personal key in the X-API-Key header with every request. Each user is assigned a unique key upon registration.

Note: In a full application, you would typically find your API key on your account settings page. For now, you would need to retrieve it directly from the database after registering an account.

Get File Content

Retrieves the content of a specific file within a bot's directory.

GET /api/bot/<user_id>/<bot_index>/file?path=<file_path>

Parameters

  • path (query string, required): The relative path to the file from the bot's root directory (e.g., main.py or cogs/music.py).

Example Request (cURL)

curl -X GET "http://your-domain.com/api/bot/USER_ID/0/file?path=main.py" \
-H "X-API-Key: your-super-secret-api-key"

Success Response (200 OK)

{
  "path": "main.py",
  "content": "import discord\n\n# Your bot code here..."
}

Edit File Content

Updates the content of a specific file. If the file doesn't exist, it will be created.

POST /api/bot/<user_id>/<bot_index>/file?path=<file_path>

Parameters

  • path (query string, required): The relative path to the file.

Request Body (JSON)

  • content (string, required): The new content for the file.

Example Request (cURL)

curl -X POST "http://your-domain.com/api/bot/USER_ID/0/file?path=config.json" \
-H "X-API-Key: your-super-secret-api-key" \
-H "Content-Type: application/json" \
-d '{"content": "{\\"prefix\\": \\"!\\"}"}'

Success Response (200 OK)

{
  "success": true,
  "message": "File 'config.json' updated successfully."
}

Start Bot

Starts the bot process.

POST /api/bot/<user_id>/<bot_index>/start

Example Request (cURL)

curl -X POST "http://your-domain.com/api/bot/USER_ID/0/start" \
-H "X-API-Key: your-super-secret-api-key"

Success Response (200 OK)

{
  "success": true,
  "message": "Bot started successfully."
}

Stop Bot

Stops the bot process.

POST /api/bot/<user_id>/<bot_index>/stop

Example Request (cURL)

curl -X POST "http://your-domain.com/api/bot/USER_ID/0/stop" \
-H "X-API-Key: your-super-secret-api-key"

Success Response (200 OK)

{
  "success": true,
  "message": "Bot stopped successfully."
}
